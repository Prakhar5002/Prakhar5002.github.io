Smart Setup — IoT Device Onboarding Platform

React Native · Expo · TypeScript · MQTT · TanStack React Query · Zustand · Keycloak

A greenfield IoT provisioning platform for smart building campuses, built entirely from scratch and now live across 60+ locations. The platform provisions Sonoff IoT switches — configuring them to the campus WiFi and registering them in the building management system — through a guided multi-step wizard designed for facilities teams.

The hardest engineering problem was connectivity: a device that loses WiFi mid-provisioning ends up in a corrupt state that's difficult to recover. I built NetInfo-based network-awareness that detects instability early and suspends the workflow before any bad state is written — this was the single biggest reliability improvement, cutting failures by ~40%. MQTT-based configuration workflows replaced entirely manual onboarding, cutting setup time by ~35% and making it possible to onboard an entire campus floor in one session. FlashList with item recycling fixed frame drops on mid-range Android for large device inventories (60%+ improvement). Auth via Keycloak OIDC/PKCE with SecureStore-backed token persistence.
